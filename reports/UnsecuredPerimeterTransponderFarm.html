Unsecured Perimeter Transponder Farm

Last edited by XeCara
Mon, 29 Apr 2013 15:46 UTC

Faction: Sleepers
Signature type: Radar
Space type: Normal Space
Damage dealt: Omni
Webbers: Emergent Escort
Scramblers: None
Recommended ship classes: 1 Ship (BC)
Recommended generic setup: Heavy omni tank + Remote repair

As with any Sleeper radar site, there are 4 waves each which auto aggress.
Sleeper ships may target switch. Drones and support ships are not immune.

5 Ejected Sleeper Databanks, which can be accessed using the Codebreaker Module.

Single Pocket


Initial Group

3 x Cruisers (Awakened Patroller) orbit 15km
1 x Frigates (Emergent Patroller) orbit 8,139m
2 x Frigates (Emergent Escort) [web] TRIGGER orbit 5,170m

Reinforcement Wave 1

1 x Cruiser (Awakened Watchman) [neut] orbit 30km
4 x Cruisers (Awakened Patroller) TRIGGER orbit 15km
2 x Frigates (Emergent Escort) [web] orbit 5,170m

Reinforcement Wave 2

1 x Battleship (Sleepless Patroller) orbit 19km
4 x Frigates (Emergent Patroller) orbit 8,139m
4 x Frigates (Emergent Escort) [web] TRIGGER orbit 5,170m

Reinforcement Wave 3

5 x Cruisers (Awakened Patroller) orbit 15km
3 x Cruisers (Awakened Escort) orbit 45km
4 x Frigates (Emergent Escort) [web] orbit 5,170m
