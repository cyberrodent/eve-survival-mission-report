Forgotten Perimeter Habitation Coils

Last edited by XeCara:

Faction: Sleepers
Signature type: Magnetometric
Space type: Normal Space
Damage dealt: Omni
Webbers: Emergent Escort
Scramblers: None
Recommended ship classes: Soloable in Passive Drake
Recommended generic setup: Omni tank

As with all sleeper magnetometric sites, there are 4 waves each which auto aggress.
Sleeper ships re-target every few minutes. Drones and support ships are not immune.

Single Pocket


Initial Group

3x Cruisers (Awakened Patroller)
2x Frigates (Emergent Escort) (web) TRIGGER
2x Frigates (Emergent Patroller)

Reinforcement Wave 1

4x Cruisers (Awakened Patroller) TRIGGER
4x Frigates (Emergent Escort) (web)

Reinforcement Wave 2

1x Battleship (Sleepless Patroller) TRIGGER
4x Frigates (Emergent Escort) (web)
4x Frigates (Emergent Patroller)

Reinforcement Wave 3

1x Battleship (Sleepless Watchman)
1x Cruiser (Awakened Patroller)
2x Cruisers (Awakened Escort)
3x Frigates (Emergent Escort) (web)

Tip
  • 5 containers to be analyzed using the Analyzer archaeology module
