Recover the Containers, Level 5


Last edited by DesmontMcCallock
Sat, 14 Jul 2012 07:20 UTC


Faction: Caldari
Mission type: Encounter
Space type: Deadspace
Web/Scramble: Stasis Towers, All Frigates 25-50% chances of Web and/or Scramble
Extras: Caldari Energy Neutralizer Sentry IIIs, ECM Jamming
Damage dealt: Kinetic 81% / Therm 19%
Recommended damage dealing: Kin/Th
Recommended ships: Completed in (1) Maelstrom w/ passive tank and (1) Maelstrom w/ 2 Shield Transporters, but recommend more DPS. Industrial needed for hauling.

Mission requires the player to be able to use a Codebreaker module for hacking

Direct warp into Deadspace. No initial gate.

Warp-In Pocket 1


Initial Group

Full Room Aggro begins 5 seconds after warp-in:

2 x Tower Sentry Caldari IIIs
4 x Caldari Cruise Missile Batteries
7 x Caldari Stasis Towers Trigger

Spawn 1

Each of the 4 closest Stasis Towers spawn one Energy Neutralizer replacement when destroyed:

4 x Caldari Energy Neutralizer Sentry IIIs


Pocket 2



Initial Group

Full Room Aggro at warp-in (Spawns appear to be timed):

2 x Cruisers (Taibu State Samurai)
3 x Battlecruisers (State Shukuro Bajo)
5 x Battleships (State Shukuro Shogun)
2 x Tower Sentry Caldari IIIs
6 x Caldari Cruise Missile Batteries

Spawn 1

3 minutes after warp-in, there spawns in the Debris Field:

2 x Frigates (Taibu State Katana/Wakizashi) Web/Scramble
3 x Battlecruisers (State Oni/Kanpaku)
2 x Battleships (State Zen)

Spawn 2

More ships spawn later in the same Debris Field:

7 x Frigates (Taibu State Katana/Wakizashi/Nagasa/Shinai/Tachi*) Web/Scramble, *ECM Jamming
2 x Cruisers (Taibu State Bajo/Samurai )
3 x Battleships (State Shukuro Daijo)

Hacking Spawns

Most of the Databanks, upon hacking, will create a spawn of:

2 x Frigates (Taibu State Wakizashi/Nagasa) Web/Scramble



Pocket 3


Initial Group

Full Room Aggro at warp-in:

4 x Battlecruisers (State Shukuro Bajo)
5 x Battleships (State Shukuro Bishamon/Daijo/Tenno/State Oshiro)
3 x Tower Sentry Caldari IIIs
2 x Caldari Cruise Missile Batteries Trigger
2 x Caldari Energy Neutralizer Sentry IIIs Trigger

Spawn 1

Cruise Missile Batteries and a Neutralizer, when destroyed, spawn a total of:

3 x Tower Sentry Caldari IIIs

Hacking Spawn

Upon Hacking the correct Databank, there spawns:

4 x Frigates (Taibu State Wakizashi/Nagasa/Shinai) Web/Scramble
5 x Cruisers (Taibu State Bajo/Samurai )


Pocket 4



Initial Group

Full Room Aggro at warp-in:

3 x Frigates (Taibu State Tachi*) Trigger 1, Web/Scramble, *ECM Jamming
3 x Cruisers (Taibu State Kanpaku) Trigger 2
(Taibu State Oni)
1 x Battlecruiser (State Shukuro Bajo) Trigger 3
6 x Battleships (State Shukuro Bishamon) Trigger 4
(State Shukuro Daijo) Trigger 5
(State Shukuro Shogun)
4 x Caldari Cruise Missile Batteries Trigger 6
4 x Caldari Energy Neutralizer Sentry IIIs

Spawn 1

Upon destroying the frigates, there spawns:

3 x Frigates (Taibu State Nagasa/Shinai) Web/Scramble

Spawn 2

Upon destroying the Taibu State Kanpaku there spawns:

3 x Cruisers (Taibu State Bajo/Samurai)

Spawn 3

Upon destroying the State Shukuro Bajo there spawns:

3 x Battlecruisers (State Shukuro Bajo/Samurai)
2 x Cruisers (Taibu State Bajo/Samurai)

Spawn 4

Upon destroying the final State Shukuro Bishamon there spawns:

3 x Battleships (State Shukuro Shogun)

Spawn 5

Upon destroying the final State Shukuro Daijo there spawns:

4 x Cruisers (State Shukuro Bajo/Samurai)

Spawn 6

As each original Cruise Missile Battery is destroyed, it spawns one replacement

4 x Caldari Cruise Missile Batteries

Tips
  • Consider FoF missiles and setting drones to “Aggressive” for when you are unable to target because of Jammers
  • Spawns more often than not will aggro secondary teammates and drones; and often split aggro
  • Each room’s Databanks contain 1 of the 3 Sealed Containers (objectives)
  • Databanks are “booby-trapped” and can explode when your Codebreaker fails on an attempt
  • Each explosion will cause 350hp of proximity damage, so plan your hacking ship for that and the hacking spawns
  • Pocket 2's maximum damage dealing of all ships: 2,130hp/s (Kin: 82%, Therm: 17%)
