Unsecured Perimeter Comms Relay

Last edited by XeCara
Mon, 07 Apr 2014 07:25 UTC

Faction: Sleepers
Signature type: Radar
Space type: Normal Space
Damage dealt: Omni
Webbers: None
Scramblers: None
Extras: NOS (Emergent Watchman, Awakened Watchman)
Recommended ship classes: Soloable in Passive Drake
Recommended generic setup: Heavy omni tank

Sleeper ships re-target every few minutes. Drones and support ships are not immune.

Single Pocket


Initial Group

3x Cruisers (Awakened Patroller)
2x Cruisers (Awakened Watchman) [neut] last is TRIGGER
1x Battleship (Sleepless Outguard) [neut]

Reinforcement Wave 1

1x Battleship (Sleepless Patroller) TRIGGER
2x Frigates (Emergent Watchman) [neut]

Reinforcement Wave 2

1x Battleship (Sleepless Watchman)
1x Battleship (Sleepless Escort)
2x Frigates (Emergent Patroller)

Tip
  • Only 3 waves in this Radar site
  • The Sleepless Outguard does not always spawn. If he does, then the Talocon Frigate will also spawn.
  • 8 containers (Ejected Sleeper Databank) to be hacked using Codebreaker module
  • Possible 1 container (Derelict Talocan Frigate) to be salvaged using Salvaging module, see Sleepless Outguard.
