Last edited by DesmontMcCallock

Faction: Sansha Nation
Mission type: Encounter
Space type: Deadspace
Damage dealt: EM, Therm
Web/Scramble: Sansha Demon/Berserker
Recommended damage dealing: EM, Therm
Recommended ships: Drake

Initial Gate

No Hostiles

Single Pocket

Spawns names may vary, this is an example.
No aggro at warp-in.
Groups aggro independently.

Group 1: (~30km)

1x Frigate class (Sansha Demon)
1x Frigate class (Sansha Berserker/Demon)
2x Cruiser class (Centum Execrator/Slaughterer)

Group 2: (55-60km)

1x Frigate class (Centii Savage)
4x Frigate class (Centii Savage/Slavehunter)
1x Cruiser class (Centum Execrator/Slaughterer)
1x Cruiser class (Centum Fiend/Hellhound)
